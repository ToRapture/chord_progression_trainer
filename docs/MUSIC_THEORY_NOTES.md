# Music Theory Notes

Function groups are training approximations:

- `T`: tonic area.
- `PD`: predominant area.
- `D`: dominant area.
- `OTHER`: borrowed, modal, secondary, or ambiguous usage.

In minor keys, `V`, `V7`, and leading-tone diminished chords are treated as harmonic-minor sonorities. This is marked in `ChordInKey.source`.

The voicing engine prioritizes clarity over stylistic realism. Root-position bass is preferred for beginner presets so bass degree recognition stays stable.
